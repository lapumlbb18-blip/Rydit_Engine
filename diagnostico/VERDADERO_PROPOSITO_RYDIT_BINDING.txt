## 📐 Arquitectura del Binding Universal (Independiente)

### Estructura Propuesta

```
shield-project/
├── crates/
│   ├── lizer/              # Lexer + Parser (core)
│   ├── blast-core/         # Executor + Memoria (core)
│   ├── rydit-gfx/          # Gráficos (core)
│   ├── rydit-rs/           # Binario principal (core)
│   └── rydit-bindings/     # ⭐ BINDING UNIVERSAL (independiente)
│       ├── src/
│       │   ├── lib.rs      # Registry de bindings
│       │   ├── scene.rs    # Bindings de escenas
│       │   ├── physics.rs  # Bindings de física
│       │   ├── gfx.rs      # Bindings gráficos
│       │   └── logic.rs    # Bindings de lógica
│       └── Cargo.toml      # Independiente del core
```

### Ejemplo de Uso

```rydit
# NO es para web, NO es para apps
# ES para simulaciones de escenas

import scene
import physics
import gfx

# Crear escena
dark.slot escena = scene::new("Mi Simulación")

# Agregar objetos
scene::add_object(escena, "pelota", {
    shape: "circle",
    radius: 50,
    color: "red",
    physics: true  # Tiene física
})

# Configurar física
physics::set_gravity(escena, 9.8)
physics::set_bounce(escena, "pelota", 0.8)

# Loop de simulación
rytmo game_loop {
    # Actualizar física
    physics::step(escena, 0.016)  # 60 FPS
    
    # Renderizar
    gfx::clear()
    gfx::draw(escena)
    gfx::present()
}
```
