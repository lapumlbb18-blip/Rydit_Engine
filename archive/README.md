# 📦 Archive - Shield Project

**Contenido:** Documentación histórica, sesiones antiguas y archivos deprecated.

## Estructura

```
archive/
├── readme-old/      # READMEs, ANUNCIOS, GUIAs deprecated
├── sessions-old/    # Logs de sesiones antiguas (SESION_*.txt, DIAGNOSTICO_*.txt)
└── docs-old/        # Documentación general deprecated
```

## ¿Qué va aquí?

- ✅ Documentación de versiones anteriores
- ✅ Logs de sesiones completadas
- ✅ Guías y anuncios deprecated
- ✅ Archivos de contexto antiguo

## ¿Qué NO va aquí?

- ❌ Documentación activa (va en `docs/`)
- ❌ Contexto de sesión actual (va en `context/current/`)
- ❌ Tests activos (van en `tests/`)

---

**Propósito:** Mantener el root limpio y encontrar rápido lo activo.
