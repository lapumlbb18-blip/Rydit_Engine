### WARN-001: unwrap() en I/O (RESUELTO)

ANTES:
```rust
io::stdout().flush().unwrap();
io::stdin().read_line(&mut input).unwrap();
```

DESPUÉS:
```rust
if let Err(e) = io::stdout().flush() {
    eprintln!("[WARNING] Error en flush de stdout: {}", e);
    return Valor::Error("Error de I/O en input".to_string());
}

if let Err(e) = io::stdin().read_line(&mut input) {
    eprintln!("[WARNING] Error al leer stdin: {}", e);
    return Valor::Error("Error de lectura en input".to_string());
}
```

### WARN-004: unused_variables (RESUELTO)

ANTES:
```rust
let start_col = column;
```

DESPUÉS:
```rust
let _start_col = column;  // Reservado para futuros errores detallados
```

### WARN-005: unreachable_patterns (RESUELTO)

ANTES:
```rust
Stmt::Block(stmts) => { ... }  // línea 136
...
Stmt::Block(stmts) => { ... }  // línea 169 (DUPLICADO!)
```

DESPUÉS:
```rust
Stmt::Block(stmts) => { ... }  // línea 136 (único)
```
