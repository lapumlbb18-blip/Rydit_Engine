================================================================================
                         💡 LECCIONES APRENDIDAS
================================================================================

1. **raylib con nobuild**: Necesita build.rs para linkear con librería nativa
2. **pkg-config**: Esencial para encontrar raylib en Termux
3. **links = "raylib"**: Necesario en Cargo.toml para linking correcto
4. **DrawHandle**: Debe usarse dentro de scope limitado para dibujar
5. **Game loop**: Input primero, luego dibujo, luego end_draw automático
