STACK DE SCOPES:
----------------
Cuando se llama a una función:

1. executor.push_scope() → Crea HashMap nuevo
2. Mapear params → arg_values en scope local
3. Ejecutar body de función
4. executor.pop_scope() → Elimina scope local

EJEMPLO DE EJECUCIÓN:
---------------------
```rydit
dark.slot x = 100

rytmo test(x) {
    dark.slot local = x * 2
    voz local
}

voz x        # Imprime 100 (global)
test(10)     # Imprime 20 (x local = 10)
voz x        # Imprime 100 (global intacto)
```

OUTPUT REAL:
```
[MEMORIA GLOBAL] x = 100
x global antes:
100
[SCOPE] Nuevo scope local creado (nivel 1)
[LOCAL] x = 10
[MEMORIA LOCAL] local = 20
20
[SCOPE] Scope local eliminado (quedan 0 niveles)
x global después:
100
```

EJEMPLO 2: Función que Modifica Variable Local
----------------------------------------------
```rydit
rytmo factorial(n) {
    dark.slot resultado = 1
    ryda n > 1 {
        dark.slot resultado = resultado * n
        dark.slot n = n - 1
    }
    voz resultado
}

factorial(5)  # Imprime: 120
```

================================================================================
                         ⚠️ LIMITACIONES ACTUALES
================================================================================

1. FUNCIONES NO SON EXPRESIONES
-------------------------------
NO SE PUEDE:
```rydit
dark.slot x = sumar(5, 3)  # ❌ Error: Función no soportada en expresiones
```

SE PUEDE:
```rydit
rytmo sumar(a, b) {
    voz a + b  # ✅ Imprime resultado
}
sumar(5, 3)
```

SOLUCIÓN FUTURA (v0.2.0):
- Implementar Expr::FunctionCall
- Retorno de valores con `return <expr>`
- Asignación: `dark.slot x = funcion()`

2. NO HAY TIPOS EN PARÁMETROS
-----------------------------
NO SE PUEDE:
```rydit
rytmo sumar(a: num, b: num) { ... }  # ❌ Sin tipos
```

SE PUEDE:
```rydit
rytmo sumar(a, b) { ... }  # ✅ Tipos dinámicos
```

3. NO HAY VALORES POR DEFECTO
-----------------------------
NO SE PUEDE:
```rydit
rytmo saludar(nombre = "Mundo") { ... }  # ❌ Sin defaults
```

SE PUEDE:
```rydit
rytmo saludar(nombre) {
    onif nombre == "" {
        dark.slot nombre = "Mundo"
    }
    # ...
}
```
