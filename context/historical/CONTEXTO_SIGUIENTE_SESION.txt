================================================================================
                         💡 LECCIONES APRENDIDAS (v0.0.7)
================================================================================

EL MURO:
--------
raylib-rs 5.5.1 tiene API inconsistente:
- Algunos métodos requieren &thread
- Otros NO lo requieren
- Signatures cambian entre versiones

LA SOLUCIÓN:
------------
Crear rydit-gfx (sync layer):
- Define constantes MANUALES para colores y teclas
- Wrapper que unifica signatures
- API consistente para RyDit
- Rust SIEMPRE controla, Raylib SIEMPRE ejecuta

PARADIGMA:
----------
Rust = Arquitecto (controla el loop, input, decisiones)
Raylib = Pincel (solo dibuja, ejecuta)
rydit-gfx = Puente (sincroniza ambos mundos)
