### PUERTA 4: Struct Builder ⭐⭐⭐⭐

```rust
// 1. Crear struct
pub struct TextureParams {
    pub id: String,
    pub position: Vec2,
    pub scale: f32,
    pub rotation: f32,
    pub color: Color,
}

// 2. Builder pattern
impl TextureParams {
    pub fn new(id: &str) -> Self { ... }
    pub fn at(mut self, x: f32, y: f32) -> Self { ... }
    pub fn scale(mut self, s: f32) -> Self { ... }
    pub fn rotation(mut self, r: f32) -> Self { ... }
    pub fn color(mut self, c: Color) -> Self { ... }
    // ← FÁCIL AGREGAR MÁS:
    pub fn flip_h(mut self, flip: bool) -> Self { ... }
    pub fn flip_v(mut self, flip: bool) -> Self { ... }
    pub fn origin(mut self, x: f32, y: f32) -> Self { ... }
    pub fn blend(mut self, mode: BlendMode) -> Self { ... }
}

// 3. Nueva firma
pub fn draw_texture(
    &self,
    d: &mut RaylibDrawHandle,
    params: TextureParams
) { ... }

// Uso:
gfx.draw_texture(&mut d,
    TextureParams::new("tank")
        .at(100.0, 200.0)
        .scale(2.0)
        .rotation(45.0)
        .color(WHITE)
        .flip_h(false)  // ← NUEVO
        .flip_v(false)  // ← NUEVO
);
```

**Consecuencias:**
- ⚠️ ROMPE TODA la API existente
- ⚠️ 8-10 horas de trabajo
- ✅ API PROFESIONAL
- ✅ Fácil agregar nuevas features

**Trabajo REAL:**
```
1. Crear struct: 15 min
2. Builder methods: 30 min
3. Cambiar firma: 15 min
4. Actualizar TODOS los calls: 2-3 horas
5. Testear TODO: 2 horas
6. Fix bugs: 2-3 horas
7. Documentar: 1 hora
TOTAL: 8-10 horas
```

**Beneficio:**
- 7 args → 1 struct
- API hermosa, legible
- Fácil agregar nuevos params
- Ej: `.flip_h(true).flip_v(true)`

**Veredicto**: OBLIGATORIO para v1.0.0
