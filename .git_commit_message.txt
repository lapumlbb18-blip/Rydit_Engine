Funciones Implementadas:
- regex::match(patron, texto) -> bool
  * Retorna true si el patrón coincide con el texto
  * Ejemplo: regex::match("\\d+", "123") -> true

- regex::replace(patron, reemplazo, texto) -> texto
  * Reemplaza todas las ocurrencias del patrón
  * Ejemplo: regex::replace("[aeiou]", "*", "hola") -> "h*l*"

- regex::split(patron, texto) -> array
  * Divide el texto por el patrón
  * Ejemplo: regex::split(",", "uno,dos,tres") -> ["uno","dos","tres"]

- regex::find_all(patron, texto) -> array
  * Encuentra todas las ocurrencias del patrón
  * Ejemplo: regex::find_all("\\d+", "a1b23c456") -> ["1","23","456"]

- regex::capture(patron, texto) -> array
  * Captura grupos de coincidencia
  * Ejemplo: regex::capture("([a-z]+):(\\d+)", "edad:25") -> ["edad:25","edad","25"]
